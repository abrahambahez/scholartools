Herramientas de investigación para trabajo entre humanos y agentes de IA

Una biblioteca Python que da a los agentes de IA una interfaz confiable y determinista para construir y mantener una biblioteca de investigación. Dos capas: un repositorio de referencias como objetos epistémicos, y una capa de conocimiento con notas de síntesis, páginas de conceptos y rutas de lectura derivadas de ellas. Local primero, sin interfaz gráfica, sin flujos manuales.

Usa agentes de IA con confianza [agent-native-functions]
Mejora la calidad de tus referencias [CSL-JSON · two-stage-model]
Explora y filtra fuentes de información [staging → library]
Convierte tu biblioteca en comprensión viva [wiki · synthesis-notes · concept-pages]
Trabaja sin conexión, controla tus datos [local-first · portable · no-GUI]

Flujo de trabajo — núcleo

          


            

              01
              
Extraer

              

                humano
                "Extrae los metadatos de este PDF"
              

              

                agente
                Listo — título, autores, año, DOI extraídos localmente. Sin red.
              

              

                cli
                lore extract paper.pdf
              

            


            

              02
              
Área de revisión

              

                humano
                "Guárdalo para revisar — no lo confirmes aún"
              

              

                agente
                Guardado en staging con el archivo adjunto. Nada en la biblioteca aún.
              

              

                cli
                lore staging stage '<json>' --file paper.pdf
              

            


            

              03
              
Fusionar

              

                humano
                "Se ven bien — agrégalos a la biblioteca"
              

              

                agente
                Listo. Duplicados eliminados, citekeys asignados, archivos organizados.
              

              

                cli
                lore staging merge
              

            


            

              04
              
Buscar y filtrar

              

                humano
                "¿Qué tengo sobre objetos frontera de 2019?"
              

              

                agente
                3 referencias. star1989 tiene archivo adjunto — ¿quieres que lo abra?
              

              

                cli
                lore refs filter --query "objetos frontera" --year 2019
              

            


            

              05
              
Archivos

              

                humano
                "Adjunta este PDF a la referencia"
              

              

                agente
                PDF archivado en vaswani2017. Vinculado y listo para leer.
              

              

                cli
                lore files attach vaswani2017 paper.pdf
