Herramientas de investigación para optimizar trabajo entre humanos y agentes de IA

Una biblioteca Python +agent skills, prompts y más artefactos que dan a los agentes de IA una interfaz confiable y determinista para buscar, obtener, almacenar y gestionar referencias académicas.

Usa agentes de IA con confianza [agent-native-functions]
Mejora la calidad de tus referencias [CSL-JSON · two-stage-model]
Explora y filtra fuentes de información [staging → library]
Comparte una biblioteca colectivamente [Ed25519 · LWW-sync]
Trabaja sin conexión, controla tus datos [local-first · small AI compatible]

Comenzar

Instala el CLI y las skills de Claude Code para empezar a usar scholartools desde tus flujos de trabajo con agentes.

macOS / Linux

Instalar CLI

curl -fsSL https://raw.githubusercontent.com/abrahambahez/scholartools/main/install.sh | bash

Instalar skills de Claude Code

curl -fsSL https://raw.githubusercontent.com/abrahambahez/scholartools/main/install-skills.sh | bash

Windows

Ejecuta estos comandos desde una terminal PowerShell abierta como Administrador (clic derecho → "Ejecutar como administrador").

Flujo de trabajo de investigación

01

Descubrir

humano "Encuentra artículos recientes sobre transformer attention"
agente Buscando en Semantic Scholar, arXiv y 4 fuentes más…
cli scht discover "transformer attention"
02

Obtener / Extraer desde PDF

humano "Obtén el registro completo para este DOI"
agente Listo. Título, autores, resumen, DOI — todo en un registro limpio.
cli scht fetch 10.48550/arXiv.1706.03762
03

Área de revisión

humano "Guárdalos para revisar antes de confirmar"
agente Guardado en staging. Nada confirmado aún — tú sigues en control.
cli scht staging list
04

Fusionar

humano "Se ven bien — agrégalos a la biblioteca"
agente Agregados a la biblioteca. Duplicados eliminados, citekeys asignados.
cli scht staging merge
05

Gestión de archivos

humano "Adjunta este PDF a la referencia"
agente PDF vinculado. vaswani2017 ahora tiene un archivo que puedes abrir o extraer.
cli scht files attach vaswani2017 paper.pdf
06

Sincronizar

humano "Sincroniza mi biblioteca con el equipo"
agente Biblioteca sincronizada. Tu equipo ve los mismos registros ahora.
cli scht sync push-changelog

Infraestructura pensada para la cambiante frontera entre la inteligencia humana y la automatización con agentes de IA.
